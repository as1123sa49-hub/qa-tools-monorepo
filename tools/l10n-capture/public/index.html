📸 L10n 擷圖 + 驗證

上傳多國語系 xlsx → 擷取遊戲截圖（階段 A）→ 以 Vision OCR 比對畫面文字是否與翻譯表一致（階段 C）。

1. 上傳翻譯表（每次執行前上傳）
拖放 xlsx 到此，或點擊選擇檔案
可複選多個工作表；每個 S0xx 對應一款遊戲（S015 → Slot015）。首次請先執行 npm.cmd run save-auth 儲存登入。
2. 擷取設定
選定一個語系後，對複選工作表對應的多個 Slot 依序擷取，產出 captures/環境/語系/SlotXXX/。語系對應大廳選單（英文、繁中、泰文…）已內建於設定檔。
3. 翻譯驗證（Vision OCR）
API Key 與模型選擇與「測試案例自動產生工具」共用本機儲存（gemini_api_key / siraya_api_key）。 會對上方 Slot 對應的 captures/環境/語系/SlotXXX 既有截圖做 OCR 與比對，無需重新擷取。 已啟用 OCR 快取（ocr-cache.json）與送圖前縮圖，重驗時可大幅節省 API 用量。PASS ≥ 95%、REVIEW 85–94%、FAIL < 85%。
