🔑 AI API 設定 Step 1
📋 取得 Gemini API Key 步驟
  1. 1 前往 Google AI Studio：aistudio.google.com/app/apikey
  2. 2 登入 Google 帳號（個人或公司帳號皆可）
  3. 3 點「Create API key」→ 選「Create API key in new project」
  4. 4 複製產生的 Key（格式：AIzaSy...），貼到上方輸入框
  5. 5 按「💾 儲存」完成，Key 僅儲存在你的瀏覽器，不會上傳
💡 免費額度依模型而異，請至 ai.dev/rate-limit 查看；多 PRD 分批建議選 3.1 Flash Lite
📋 取得 Siraya API Key 步驟
  1. 1 前往 Siraya：siray.ai 註冊帳號
  2. 2 至設定頁建立 API Key
  3. 3 複製 Key 貼到上方輸入框，按「💾 儲存」
💡 Siraya 支援 Gemini、GPT、Claude 等模型，按用量計費；端點為 OpenAI 相容 API（llm.siraya.ai）
