# Bonus V2 電子骰機率驗證 — Chrome Extension

不需要安裝 Node.js，純 Chrome Extension，打開遊戲頁面後自動運作。

## 安裝步驟（只需做一次）

1. 打開 Chrome，網址列輸入：`chrome://extensions`
2. 右上角開啟 **「開發人員模式」**（Developer mode）
3. 點左上角 **「載入未封裝項目」**（Load unpacked）
4. 選擇本資料夾（`bonus-v2-extension/`）
5. 安裝完成，Extension 清單中會出現「Bonus V2 電子骰機率驗證」

## 使用方式

1. 打開遊戲頁面（網址含 token）
2. 頁面右上角會自動出現統計面板
3. 等待電子開牌（投注時間結束後）→ 資料自動收集
4. 點「📊 統計 CSV」或「📋 明細 CSV」匯出資料

> ⚠️ 如果進入遊戲頁面時面板沒有出現，請按 F5 重新整理一次

## 面板說明

| 功能 | 說明 |
|------|------|
| 拖曳標題列 | 移動面板位置 |
| 「—」按鈕 | 最小化 / 展開 |
| 差值顏色 | 🟢 < 5% 正常、🟡 5~10% 留意、🔴 > 10% 偏差 |
| 📊 統計 CSV | 匯出倍率分布對比表（含期望% vs 實際%） |
| 📋 明細 CSV | 匯出每局原始資料 |
| 清空 | 重置所有收集資料 |

## 資料夾結構

```
bonus-v2-extension/
  manifest.json    ← Extension 設定
  content.js       ← 注入用腳本（極簡）
  page-script.js   ← 主邏輯（WS 攔截 + 統計面板）
  README.md        ← 本說明檔
```
